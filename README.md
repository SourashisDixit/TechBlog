    <h2>&#128218; About the Project</h2>
    <p>
        <strong>TechBlog</strong> is a web application that allows users to read, write, and share technical articles easily.<br>
        It's designed to promote knowledge sharing among developers through an interactive, user-friendly interface.<br>
        The project demonstrates strong backend development skills using <strong>Core Java, Servlets, JSP, and JDBC</strong>.

        <li>&#129521; View Blogs by Category or Author</li>
        <li>&#128172; Comment Section for Readers</li>
